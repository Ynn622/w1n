# 即時風況 App - 專案說明

## 📱 應用概述

「即時風況」是一個整合即時天氣、風況、路況資訊的 Web 應用程式，提供便捷的路線規劃與氣象查詢功能。

## 🎨 介面設計

### 介面 1：首頁（主功能入口）
**路由：** `/` (home)

**版面結構：**
- 上方區塊（20%）：App 標題「即時風況」
- 中間功能區（60%）：三個主要功能按鈕
  - 🌪️ 即時風況
  - 🚗 即時路況
  - 📰 新聞
- 下方導航（20%）：底部導航列

**檔案位置：** `src/views/Home.vue`

---

### 介面 2：即時路況（路線規劃）
**路由：** `/traffic` (traffic)

**版面結構：**
- 上方搜尋欄（25%）：出發地與目的地輸入
- 中間路線區（60%）：路線規劃結果顯示
- 下方導航（15%）：底部導航列

**主要功能：**
- 輸入起點終點搜尋路線
- 顯示多條建議路線（時間、距離）
- 右下角 Google Map 轉跳按鈕

**檔案位置：** `src/views/Traffic.vue`

---

### 介面 3：風況詳細（天氣資訊牆）
**路由：** `/wind` (wind)

**版面結構：**
- 上方功能列（15%）：App 標題
- 中間風況區（50%）：
  - 風況詳細資訊（風向、風速、溫度、濕度、氣壓）
  - 篩選功能（日期、時間、地區）
- 下方新聞牆（35%）：氣象相關新聞列表

**主要功能：**
- 即時風況儀表板
- 風向視覺化指示器
- 氣象數據展示
- 新聞資訊卡片

**檔案位置：** `src/views/Wind.vue`

---

### 介面 4：個人設定（偏好與常用路線）
**路由：** `/settings` (settings)

**版面結構：**
- 上方標題列（15%）：「偏好設定」
- 中間設定區（70%）：
  - 常用路徑管理（新增、刪除）
  - 通知設定
  - 地圖顯示偏好
  - 介面主題選擇（亮/暗/自動）
- 下方導航（15%）：底部導航列

**主要功能：**
- 常用路線的增刪管理
- 通知開關設定
- 地圖應用選擇
- 主題模式切換

**檔案位置：** `src/views/Settings.vue`

---

## 🧭 導航結構

### 底部導航列
固定於所有頁面底部（高度約 12-15%），包含四個按鈕：

1. 🏠 **首頁** → `/`
2. 🚗 **即時路況** → `/traffic`
3. 🌪️ **風況資訊** → `/wind`
4. ⚙️ **偏好設定** → `/settings`

**檔案位置：** `src/components/BottomNav.vue`

---

## 🎨 設計規範

### 色彩配置
- **主色調：** 藍色系（Primary Blue）
- **首頁：** 藍色漸層背景
- **路況頁：** 綠色主題
- **風況頁：** 藍青色漸層
- **設定頁：** 紫粉色漸層

### 高度比例
- **標題列：** 10-20% 視各頁面需求
- **主內容區：** 50-70%
- **底部導航：** 12-15% 固定

### 響應式設計
- 使用 Tailwind CSS 的響應式工具類
- 支援行動裝置優先設計
- 格子佈局適配桌面版

## 🚀 快速開始

```bash
# 安裝依賴
npm install

# 開發模式
npm run dev

# 建置生產版本
npm run build
```

## 📱 功能特色

✅ **即時風況監測** - 風速、風向、溫度、濕度、氣壓  
✅ **智慧路線規劃** - 多條路線建議、時間距離估算  
✅ **氣象新聞** - 即時氣象相關新聞推送  
✅ **常用路線** - 儲存常用路徑快速存取  
✅ **個人化設定** - 通知、地圖偏好、主題切換  
✅ **Google Maps 整合** - 一鍵轉跳導航  

---

## 🛠️ 技術棧

- **框架：** Vue 3 + TypeScript
- **路由：** Vue Router
- **樣式：** Tailwind CSS
- **組件：** 自訂 Base 組件系統
- **建置工具：** Vite
